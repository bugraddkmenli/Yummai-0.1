# Onboarding Illustrations

This directory contains custom illustrations for the onboarding screens of the Yummai application.

- `learn_to_cook.png` - Illustration for the first onboarding screen (Learn to Cook)
- `master_chef.png` - Illustration for the second onboarding screen (Become a Master Chef)
- `personalized_experience.png` - Illustration for the third onboarding screen (Personalized Cooking Experience)

These illustrations replace the placeholder images to create a more engaging and cohesive onboarding experience.
